# Ejemplo con Selenium IDE
[![GitHub issues](https://img.shields.io/github/issues/ualhmis/seleniumLoginClase.svg)](https://github.com/ualhmis/seleniumLoginClase/issues)
1. Cargar la Test Suite con Selenium IDE (plugin de Firefox) o alternativamente en Katalon Recorder
  * Requisitos: Tener instalado el plugin  Selenium IDE o Katalon Studio (explicado aquí: https://github.com/ualhmis/seleniumIDEejemploClase)
    * Si usas Katalon Recorder cargar el test suite: _LoginHmis2018.html_
    * Si usas Selenium IDE for FF <54, cargar el test suite: _loginhmis_TS.html_
  
2. Lanzar los tests (sobre https://loginhmis2018vs2017.azurewebsites.net) (Actualizado abril 2018)

3. Añadir los tests necesarios para probar toda la funcionalidad de la aplicación: 
  * Registro de usuario (Caso correcto - hecho)
  * Registro de usuario (Casos incorrectos - por hacer)
  * Login de usuario (Caso correcto - hecho)
  * Login de usuario (Casos incorrectos - por hacer)
  * Cambio de contraseña de usuario (Caso correcto - por hacer)
  * Cambio de contraseña de usuario (Casos incorrectos - por hacer)
